# C++ Object-Oriented Programming & Data Structures – Academic Exercises

This repository contains a collection of academic exercises in C++ focusing on fundamental and intermediate concepts in Object-Oriented Programming, data structures, and core programming paradigms.

The assignments were completed as part of university coursework in Computer Science fundamentals and aim to demonstrate strong understanding of software design principles and low-level programming concepts.

## 📚 Topics Covered

The exercises include implementation and practice of the following topics:

- Binary files handling and file I/O operations
- Classes with dynamic memory management
- Inheritance and polymorphism
- Operator overloading
- Virtual functions and runtime polymorphism
- Exception handling mechanisms
- Static functions and class-level design
- Data structures: Stack and Queue implementations
- Standard Template Library (STL) usage
- Linked lists and dynamic data structures

## 🧠 Learning Objectives

The purpose of these exercises is to strengthen:

- Understanding of Object-Oriented Design principles
- Practical implementation of memory management in C++
- Ability to design and implement abstract data structures
- Familiarity with standard libraries and efficient coding practices
- Problem-solving using structured programming techniques

## 🛠 Technologies

- C++
- Standard Template Library (STL)
- Object-Oriented Programming paradigms
